# The Protocol - Interactive Journey

An interactive web experience that transforms a comprehensive guide on behavior change, psychology, and goal-setting into an engaging, guided journey.

## Features

- **Monochrome Design**: Stark black and white aesthetic with monospace fonts for focused, distraction-free engagement
- **Progressive Journey**: Four main sections guiding users through conceptual understanding to actionable planning
- **Free Navigation**: Jump between sections anytime via top navigation
- **Auto-Save**: All responses automatically saved to browser localStorage
- **Browser Notifications**: Timed reminder system for throughout-the-day reflection prompts
- **Export Options**: Download complete protocol responses as both PDF and plain text
- **Word Count Tracking**: Real-time feedback on response depth
- **Expandable Concepts**: Seven foundational ideas explained in detail
- **Three-Part Protocol**:
  - Morning: Psychological excavation (14 deep questions)
  - Day: Scheduled interrupts for pattern-breaking reflection
  - Evening: Synthesis and goal-setting
- **Game Plan Framework**: 6-component life design system

## Usage

1. Open `index.html` in a modern web browser
2. Complete the commitment check to begin
3. Explore the seven foundational concepts
4. Work through the three-part protocol
5. Create your game plan
6. Export your responses for future reference

## Technical Stack

- Vanilla JavaScript (ES6+)
- CSS3 with Grid/Flexbox
- HTML5 Notifications API
- localStorage for persistence
- jsPDF for PDF export
- No external dependencies (except jsPDF CDN)

## Design Philosophy

- Minimal, monochrome aesthetic
- Zero distractions
- Intentional friction for deeper engagement
- Privacy-first (all data stored locally)
- Mobile-responsive layout

## Browser Compatibility

Requires a modern browser with support for:
- ES6+ JavaScript
- CSS Grid and Flexbox
- localStorage API
- Notifications API (optional, for reminders)
